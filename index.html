Política de Privacidade

Calculadora de Juros Compostos

1. Informações que Coletamos

O aplicativo Calculadora de Juros Compostos foi projetado com privacidade em mente. Aqui está o que coletamos:

Importante: Não coletamos dados pessoais identificáveis. Todas as informações financeiras que você insere permanecem apenas no seu dispositivo.
  • Dados de uso: Informações anônimas sobre como você usa o aplicativo (para melhorias)
  • Dados de compras: Informações de transações processadas pela Google Play
  • Dados de anúncios: Informações fornecidas pelo Google AdMob para exibir anúncios relevantes

2. Como Usamos as Informações

Utilizamos as informações coletadas para:

  • Melhorar a experiência do usuário
  • Processar compras in-app
  • Exibir anúncios relevantes (versão gratuita)
  • Garantir o funcionamento correto do aplicativo

3. Compartilhamento de Dados

Não vendemos suas informações pessoais para terceiros.

Podemos compartilhar dados apenas com:

  • Google AdMob: Para exibir anúncios (versão gratuita)
  • Google Play: Para processar compras in-app
  • Serviços de análise: Dados anônimos para melhorias

4. Segurança

Implementamos medidas de segurança para proteger suas informações:

  • Todos os dados financeiros ficam armazenados apenas no seu dispositivo
  • Utilizamos cripticação para transações
  • Não acessamos seus dados financeiros pessoais

5. Seus Direitos

Você tem direito a:

  • Acessar seus dados pessoais
  • Solicitar a exclusão de seus dados
  • Cancelar compras in-app conforme políticas da Google Play
  • Desativar anúncios mediante pagamento

6. Crianças

O aplicativo não é direcionado a crianças menores de 13 anos. Não coletamos intencionalmente informações de crianças.

7. Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos você sobre alterações significativas através do aplicativo.

8. Anúncios

O aplicativo utiliza o Google AdMob para exibir anúncios. O AdMob pode:

  • Usar cookies e tecnologias semelhantes
  • Coletar informações sobre seu dispositivo
  • Exibir anúncios personalizados

Você pode desativar anúncios personalizados nas configurações do seu dispositivo ou mediante pagamento no aplicativo.
